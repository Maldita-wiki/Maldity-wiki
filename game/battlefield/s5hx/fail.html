S5HX 错误提示

本页说明 S5HX 登陆时遇到的错误提示以及解决方法

  • 1. 登录链接错误/操作超时
  • 1.1 Download error!
  • 1.2 注入卡在 download started

网络不好的务必自己使用科学上网 如果科学上网不起作用 请使用 3A 加速器 加速瓦洛兰特 全区服

网络问题，使用科学上网

网络问题，同理！

  • 2. 远程服务器返回错误：(401) 未经授权

远程服务器返回错误：（401）未经授权

检查账户名和密码是否正确： 账户：s5hx 密码：shinku5_0237

  • 3. 连接被意外关闭

点击 download 后提示连接被意外关闭 关闭 use test version，尝试使用使用 3A 加速器

  • 4. Download 后程序未响应

download 后程序未响应 未响应是代表程序在从服务器下载数据 使用 3A 加速器或者科学上网

  • 5. 注入后游戏闪退

注入后游戏闪退

1. 安装微软运行库，重启电脑 2. 使用 3A 加速器（加速瓦洛兰特） 或者科学上网 3. 关闭 Use test version，再次 Download 下载注入 4. 是否更换过电脑？重装系统？更换 HWID？ 如果有则联系卖家解绑 5. Key 是否已经过期（key invalid, key block）？过期则需要续费 6. 不使用 Auto inject，先进入游戏，后运行启动器，游戏打开后，点击 Download and Inject 7. 系统必须为 Win10 及以上，最好更新到 21H1 或者 21H2 8. 关闭 BIOS 中的 Secure Boot。 9. 关闭内核隔离：打开 Windows 安全中心，选择设备安全性，选择内核隔离，关闭内存完整性 10. 重装系统

注意

若之前注入成功过，后来又无法注入（闪退），请查看 我的文档/s5/bf 文件夹中的 hack 文件，打开，拉到最下面，看注入失败时，提示什么？

提示 1：ACTIVATION_NO_CONNECTION ！error 1 解决：网络问题；建议使用 3A 加速器或者 VPN 全局模式

提示 2：12029 err 解决：VPN 或者加速器节点有限制，更换节点或者线路

提示 3： !AL ACTIVATION_ALREADY_USED 解决：Key 已经在其他电脑激活，需要解绑

提示 4：！AL t_e 解决：改过系统时间。将时间同步到互联网

提示 5：ACTIVATION_BAD_CODE 解决：你的 key 和你选的游戏不匹配，或者 key 中有空格

  • 6. 注入后没反应，没菜单/注入失败

1. s5hx 程序设置管理员启动 2. 游戏设置管理员启动（硬盘根目录设置，Steam 版本不需要，否则有可能会导致游戏无法打开） 3. 杀毒软件关闭（包括 Win10/Win11 自带），或者在其他杀毒软件中把 S5HX 添加白名单，并关闭防火墙 4. 开启 HTTP2, TLS。打开控制面板，找到 Internet 选项，在 Internet 选项界面的“高级”选项卡“设置”框中按下图设置后应用确定即可

  • 7. 注入后严重掉帧

1. 关闭 ESP 中的部分功能，有些系统版本会导致透视掉帧 2. 重装系统

  • 8. Win7 用户在登录软件时出现 The request was aborted: Could not create SSL/TLS secure channel

升级 Win10 或者 Win11

  • 9. 启动战地时出现 Fatal Error

启动战地时出现 Fatal Error 重启电脑后再尝试启动游戏

  • 10. bfv.exe - 损坏的映像

重启电脑后再尝试启动游戏

  • 11. Updating to a newer version failed

网盘内下载最新注入器

  • 12. 启动失败提示
  • 12.1 文件验证失败

删除你的 S5 文件夹与注入器文件夹并重启电脑，然后重新按照教程注入

删除游戏目录里除 DATA 以外的全部文件，然后验证游戏完整性

  • 13. Driver load error: c0000022
  • 13.1 Driver load error: c0000428

杀毒问题，详情查看教程

重启电脑并关闭 secure boot

  • 14. 游戏崩溃/卡住

游戏设置窗口模式

  • 15. 压缩包文件问题

将压缩包的内两个文件全部解压并放在同一个文件夹内

  • 16. 解压错误

  • 17. 提示不支持 EACC 版本
  • 17.1 有注入成功提示但是进游戏闪退

对于不支持 EAAC 版本的用户，请确保你已解压缩包并提取了同一文件夹中的所有文件，如果已经这样做，请重新启动电脑

删除 c:/windows/system32/drivers/etc/hosts 后再次尝试

  • 18. 激活码错误

确保你下载的注入器是最新版本的或检查自己激活码有没有复制错误

  • 19. 杀毒软件冲突

关闭并卸载你的杀毒以及卸载带有反作弊的游戏，例如：瓦
