XONE 疑难杂症

本页说明如修改 XONE 疑难杂症

  • 1. 注入错误

如果你注入时出现错误，打开你下载的软件压缩包，找到“Распространяемые пакеты”文件夹，在这个文件夹中具有一些依赖库，安装它们再试。

  • 2. 注入缓慢问题

你可以尝试用加速器进行加速

如果你有 Clash 或者其他的全局代理软件，那么你可以这样做 顺序如下： 1. 如果你要使用加速器，那么先打开加速器但不要加速，放在一旁 2. 打开你的代理软件并全局代理，请注意是全局 3. 登录账号，进行注入，提示后开启游戏 4. 当开启游戏后等待注入完成，确保菜单已经完全注入 5. 关闭全局代理，打开加速器

  • 3. 安全性问题
  • 3.1 外部注入安全性

我一直在使用内部注入，也许你可以自行尝试外部注入

在我测试的期间，国服与国际服官匹看来没有任何问题，Xone 的表现十分安全（内部注入） 但对于 5E 完美世界 等其他竞技平台，一些日常的操作也许会非常危险

目前我只对 完美世界竞技平台 进行测试，在我的测试情况中，你必须要尽可能避免使用 Insert 按键打开菜单，平台会检测这一行为并且快速的封禁你的账号

如果你在 完美世界竞技平台 中，在调整菜单配置时被封禁，那么非常可能是上述原因。如果你避免这个问题，只在开始按一次 Insert 快速调整配置，那么将会非常安全 如果你被检测并封禁，这个封禁可能是一天，下次注意即可。其他平台尚未测试

在外部注入期间，经测试：国服与国际服官匹没有任何问题，目前十分安全！ 但对于 5E 完美世界 等其他竞技平台，在我的测试情况中，你必须要尽可能避免使用 Insert 按键打开菜单，或用其他按键频繁调整参数，平台会检测这一行为并且快速的封禁你的账号 如果你在 完美世界竞技平台 中，在调整菜单配置时被封禁，那么非常可能是上述原因 如果你被检测并封禁，这个封禁可能是一天，但也可能有追加，下次注意即可。其他平台尚未测试

  • 4. 128 / 64 Tick 切换
  • 4.1 切换说明

CS2 官匹为 64 Tick，如果你需要 128 Tick，那么你可以这样做

切换： 1. Xone 默认会把一些文件保存在 C:\XONE 路径下，找到它，并进入 Records 文件夹 2. 找到 maps#de_mirage.bsp 文件夹，删除它 3. 打开你下载的软件压缩包，找到“Мувы на 128 тикрейт”文件夹 4. 复制 maps#de_mirage.bsp，将其粘贴到 Records 文件夹内

现已不支持 64/128 Tick 切换

  • 5. 如果被 VAC
  • 5.1 清除痕迹功能说明

我不太认为 Xone 会被检测到，也许它会在刚更新的一段时间内不安全，你应该自行排除被人工封禁的可能（增强你的演技）

当你被 VAC，尽管 VAC 没有机器码封禁，但如果你被封禁，VAC 会在你的电脑上留下痕迹，在你使用其他账户时遭到更严格的检查。通常把这一情况称之为 信任因素 Xone 提供了清除痕迹的功能： 1. 打开 Xone，但不要注入 2. 点击左下角 Settings 选项 3. 依次选择 Remove traces, Reset Activity, Reset Trust 4. 重启

我不太认为 Xone 会被检测到，也许它会在刚更新的一段时间内不安全，你应该自行排除被人工封禁的可能（增强你的演技） Xone 在几次更新后去除了 Remove traces, Reset Activity, Reset Trust 功能 也许只需要增强你的演技 排除你被人工封禁的可能后，你就需要改进你的演技了~

  • 6. 注入错误提示
  • 6.1 网络问题

服务器问题，等待修复即可

下载 everything 后把 XONE 的文件全部删除干净后，再下载注入器重新尝试注入

  • 7. 注入后切换中文乱码

没有语言文件，删除 C 盘 XONE 文件夹后重新注入

  • 8. 找不到.dll

安装 DirectX 运行库

配置调优​

  • 1. 目标选择

通常对于休闲竞技等有明显团队区分的模式，自瞄使用 仅可见与仅敌人 对于团队死斗则需要取消 仅敌人 选项，透视也一样

  • 2. 细节调整
  • 2.1 更新说明

首先根据图片依次点开

现在你可以看到三个配置项：Movement scale / Update TPS limit / FPS 限制 需要知道的是，Xone 的自瞄工作方式为：在指定范围内按或按住左键，Xone 会调整你的准星位置达成自瞄效果

在几次更新后，Xone 去除了 Movement scale 功能，只有 Update TPS limit / FPS 限制 这同样会影响到你的自瞄质量，同时也会影响到透视的更新速度。将其更改为你的显示器最大帧率即可。如果你有游戏限制帧率，那么更改为你限制的最高帧率 如果你的帧率在注入后会变低，那么可以适当降低这两值，我推荐最低不小于 60 这两个值只推荐为游戏限制的最高帧率或显示器最大刷新率，过高是消耗性能且没有意义的，这会导致你的自瞄速度起飞或非常卡顿

  • 3. Movement scale

该选项越高，你的自瞄看起来越快，越容易暴露，如果你需要调整自瞄的速度，那么去自瞄内调整，除非你认为自瞄内的范围已经不够，可以适当增高或降低该值，默认即可

  • 4. Update TPS limit / FPS 限制

这同样会影响到你的自瞄质量，同时也会影响到透视的更新速度。将其更改为你的显示器最大帧率即可。如果你有游戏限制帧率，那么更改为你限制的最高帧率 如果你的帧率在注入后会变低，那么可以适当降低这两值，我推荐最低不小于 60 警告：你不能把这两个选项拉满，如果你自己无视风险

  • 5. 自动压枪
  • 5.1 自动压枪说明

暂时不推荐开

Xone 的自动压枪看起来只在瞄准到敌人并且启用自瞄进行攻击时才会触发，效果很明显，简易开启并保持默认配置

  • 6. 延迟
  • 6.1 延迟说明

这似乎是如果两个人离的很近，自瞄目标切换的延迟。如果你真的会一直按住左键来玩这个游戏，那么将其调高，这有利于你配合演技

两个人离的很近时自瞄目标切换的延迟。如果你真的会一直按住自瞄按键来玩这个游戏，那么将其调高，这有利于你配合演技

  • 7. 目前已知 Bug

Xone 的仅可见选项有时会无效，尽管隔着一堵墙 Xone 依然会瞄准对方，可能可以穿死对方，但这绝对不利于你演戏 个人推荐将自瞄按键设置为 鼠标侧键 或 Alt 等按键，不要每时每刻都按住自瞄键
