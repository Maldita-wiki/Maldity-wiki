注入失败或游戏闪退解决手册

请根据你的游戏类型调整,重装系统是最后的方法

前言​

  • 为了避免浪费时间,先查看菜单状态

  • 确保菜单是否可用

  • 🌎菜单状态
  • 每个人都可能或多或少的遇到一些问题,有些问题可能取决于你系统,有些问题源于菜单本身,而且不同游戏遇到的问题也不尽相同
  • 为了保证大家都能尽可能的解决问题,会以 普遍解决方案 - 游戏设定 - 菜单本身解决方案 三个方面去撰写 (网页上的问题请看 网络问题
  • 此界面上部分列出的为 普遍解决方案 ,点击教程网左侧每个游戏或产品分类可查看到 游戏设定 , 菜单本身解决方案 会写在每个菜单的 使用向导 与 疑难杂症 中
危险

本店不再提供任何 Win10 以下版本的教程,教程写的全系统也指的是 Win10/11；若你在使用网吧/办公室公用电脑,有些问题可能无法解决作,请去个人电脑上操作 (鉴于网吧/办公室环境特殊,每个网吧都不尽相同,而且因为重启就恢复设置的设定导致很多东西装不了,因而没办法保证可用性[如果实在要在网吧用建议选可试用的或购买期限卡])

解决顺序​

1.购买前请确认产品必备运行环境是否具备,产品是否可用(比如游戏刚大更新肯定是用不了的),卡密是否正确；尝试重进程序/重启电脑等重试几次；若英文报错请尝试截图/拍照翻译自行尝试解决

2.若出现提示,请查看是否是 常见错误提示 的内容

3.点击教程网左侧游戏分类(若只是展开分类没有出新页面即跳过此步),完成里面的操作

4.仔细浏览所使用产品的 使用向导,确保操作正确；查看 单独的疑难杂症篇(若有),对照是否是你遇到的问题,若是就完成里面的操作

5.排查 必备环境操作中的所有,并完成里面的操作(建议所有人认真执行一遍)

6.排查 网络问题 中的所有,并完成里面的操作

7.排查 可选操作 中的所有,并完成里面的操作 (这一部分一般遇到的概率低,有的可能只适用于个别产品,我尽量写在使用向导中)

8.以上无法解决,尝试联系售后

信息

联系售后时请详细说明你用的什么菜单,发生了什么现象(最好带图/视频)

9.还不行,尝试 重装系统

必备环境​

1.完整修复运行库

2.同时按下 Win键+R (Win 键一般在键盘左下角 Ctrl 和 Alt 中建那个,窗口图标),输入 control,回车,点击 “程序”,点击 “启用或关闭 Windows 功能”,勾选 “.NET Framework 3.5 (包括 .NET 2.0 和 3.0)”,确定即可

3.关闭所有杀毒软件(若你不放心,可以安装一个火绒,对于菜单的误报率会低些)

4.关闭防火墙：同时按下 Win键+R,输入 control,回车,点击 “系统和安全”,点击 “Windows (Defender) 防火墙”,点击左侧 “启用或关闭 Windows (Defender) 防火墙”,两个都关闭,确定

5.确保运行任何 exe 应用程序的时候是以：右键 - 以管理员身份运行 的

6.关闭第三方反作弊(若你没装 Valorant 请无视此条) (Valorant/Faceit),点击这里 进入文件夹,下载运行

可选操作​

  • 系统/硬件/断网/语言区域时间问题
  • 游戏环境问题
  • 其他

1.确保你的系统非精简版/LTSC/最新预览体验版本/早期win10|11系统,若是,建议 重装系统

2.确保你的 CPU 没有过于老旧 (一般三代酷睿及以前的可能会[14 年前]),因为有些产品用的指令集太老的 CPU 没有,这种只能换 CPU 来解决

3.确保未在虚拟机/云服务器环境下运行 (若不懂请无视此条)

4.确保同步系统时间：右键 左下角开始菜单 - 时间和语言 - 日期和时间 - 勾选 “自动设置时间” (若勾选了就取消并重新勾选一次)

5.确保时区是 UTC+08:00 北京… (部分国内菜单有此要求),方法同上：日期和时间 内

6.关闭 Hyper V 虚拟机：Win+R,输入 cmd 回车,输入下方的并确定

bcdedit /set hypervisorlaunchtype off

7.确保网络是不是时断时续 (比如你正常游戏会不会经常掉线,部分产品用严格心跳检测,检测到你突然断了那么使用过程中就会闪退/失效)

8.确认区域是否正确 (Win11为左下角开始菜单 - 设置 - 右上角区域点进去 - 区域 - 国家或地区是否为中国,区域格式是否为中文(简体, 中国) (这个一般现象为出现 ??? 乱码或是程序初始化验证不通过)

9.未更改进程优先级 (若不懂不用管这一项)

10.确保 Windows 用户名未含特殊字符 (中文等,建议纯英文+数字) (主要影响那些安装目录在 %appdata%/文档 的国外菜单),若有：参考 修改系统用户名为英文

11.卸载最新的 Windows 补丁 (左下角开始菜单 - 设置 - Windows 更新 - 更新历史记录 - 卸载更新) (卸载哪个呢？若之前能用突然不能用了看下有没有最新更新的补丁,若你第一次用请尝试删除菜单更新日期之后的补丁[尤其是用于 Microsoft Windows 安全更新/更新],这个解决方案请在所有方案都试了无果的情况下再试)

1.验证游戏完整性 (若本来游戏正常运行,请无视此条；Steam 为 - 右键游戏 - 本地 - 验证完整性,其他平台百度如何操作)

2.确保未启用任何游戏内叠加层,包括 游戏加加、KOOK、Discord、Steam、Xbox Game Bar 或 NVIDIA GeForce Experience 等 (这个一般是注入时无法绘制界面的现象)

3.确保未安装模组 (Reshade 画质模组等)

4.尝试窗口化运行游戏再注入

1.确保程序未在含中文的路径下运行的 (国外产品为主)

2.对于注入式的菜单,请确保未同时开启了其他菜单

3.关闭第三方调试器 (CE 等),并确保后台干净 (关闭所有你能关的),可以尝试 Ctrl+Alt+Del 任务管理器,找到启动/启动应用,禁用你用不上的(看不懂的可百度) 并重启一次

网络问题​

  • 若你遇到无法访问/连接失败/太卡等),请看此篇；此教程仅限于菜单服务器正常的情况,若云端服务器出现问题你再怎么弄也是多余 (有的菜单教程都有 可用状态 这部分,先判断是否是他们服务器的问题)
  • 下面可能有点复杂,因为分成了很多种情况,但请耐心看完,对你今后自己处理也有很大帮助

  • 雷神加速器
  • 薄荷加速器
  • 鲜牛加速器
  • TX 加速器

  • UU：路由模式/模式 3 (UU/奇游相对最稳)

  • 迅游：模式 3 (老牌也不错)
  • AK：模式 5 (自用推荐,性价比高,时长计算,不行用 模式 4)

网络修复​

  • 若你电脑可连接手机热点,连接
  • 若方便重启路由器,重启
  • 若完全无法访问/无法连接 (尤其是国内菜单),请 点击这里,进入 文件夹 - 下载 网络修复 ComIntRep.zip 并完整解压
危险

若无法下载或解压 请确保已关闭所有,若不会请看 关闭所有杀毒软件 若不会解压请看 如何解压文件

小贴士

永远养成一个好习惯:下载任何程序不要下到含 中文/中文符号/奇怪字符 的文件夹中,这可能导致部分程序运行不了;并建议将单个程序/压缩包放入单独的文件夹中,防止部分程序释放出很多闲杂文件或解压出太多文件影响整洁度

对着解压出的 ComIntRep_X64 - 右键 - 以管理员身份运行 ,不管其他的直接点击 开始 ,耐心等待,完成后重启即可
