懒人方案解决90%系统设置问题

指令使用 ChatGPT / DeepSeek 与xk整合而来,转载请保留！

指令说明​

此指令包含：

关闭 Windows 安全中心里的大部分功能；

禁用 Edge 浏览器下载检查；

关闭快速启动；

关闭虚拟机；

设置系统时间同步；

检查用户名是否为中文；

检查 BIOS 是否为 UEFI 等等(此代码用来解决大部分设置问题)；

报红也无所谓；

在此基础上再查看本教程网的彻底关闭杀毒 疑难杂症等解决方式

执行指令

鼠标点击下方代码块下拉,点击右侧 粉色按钮 - 复制到剪贴板,

右键 点击左下角 Windows 开始菜单 - 点击 终端管理员(Windows PowerShell [管理员]),

右键点击黑色窗口 - 仍然粘贴,再按下回车,之后关闭即可 (包含禁用edge浏览器的下载检查,手动关闭浏览器生效)
